R package TreatmentSelection

evaluate biomarkers used to guide patient treatment decisions

Download package as a .zip file Download package as a tar.gz file

An R package containing a suite of descriptive and inferential methods designed to evaluate individual treatment selection markers and to compare candidate markers.

functions included are:

  • trtsel for creating trtsel objects
  • plot.trtsel for plotting risk curves and more
  • eval.trtsel for evaluating marker performance
  • calibrate.trtsel for assessing model calibration
  • compare.trtsel to compare two trtsel objects.

Please download the current version of the package as a .zip (for windows) or as a .tar.gz (for unix) using the links above.

Alternatively, to download and install the package directly from github, type:

if (!require("devtools")) install.packages("devtools")
devtools::install_github("TreatmentSelection", "mdbrown")
